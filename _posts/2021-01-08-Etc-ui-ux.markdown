# UI (User Interface)

---

- 시각화되는 작업
- 사용자가 서비스와 상호작용할 수 있도록 도와주는 매개체

# UX (User Experience)

---

- 사용자 중심적으로 설계된 디자인
- 사용자들의 경험을 분석하여 더 효율적인 방향으로 프로세스가 진행될 수 있도록 하는 과정

# UI와 UX의 차이점

---

UI는 시각적으로 표현되는 것이라면 UX는 사용자를 중심으로 고려하여 설계하는 것
