### 필요한 파일

-   node.js
-   npm
-   npx
-   create-react-app(페이스북에서 만든 react 웹 개발용 boilerplate)

-v 로 설치가 되었는지 확인

### 파일 설정

```
npx create-react-app (폴더이름)
```

### 실행하는 방법

```
npm run start 또는 npm start
```

### 종료하는 방법

Ctrl+C

<br>

### 빌드하는 방법

```
npm run build
```

### 빌드 파일 실행하는 방법

```
npx serve -s build
```
